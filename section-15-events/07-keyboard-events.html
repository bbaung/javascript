text

Modifier keys

  • event.shiftKey → true if Shift is pressed.
  • event.ctrlKey → true if Ctrl is pressed.
  • event.altKey → true if Alt (or Option on macOS) is pressed.
  • event.metaKey → true if Meta (Windows key / Command on macOS) is pressed.

keyboard events

  • keydown
  • keypress
  • keyup
  • event.key
  • event.keyCode
  • event.location
  • 0 for default keyboard,
  • 1 for left location (such as the left Alt key),
  • 2 for right location (such as the right Shift key),
  • 3 for the numeric keyboard,
  • 4 for mobile (indicating a virtual keypad), or
  • 5 for joystick (such as the Nintendo Wii controller)

#regular expression

meta-character

single character

Character Description
. Matches any single character except newline
\d Matches any digit (0-9)
\D Matches any non-digit character
\w Matches any word character (alphanumeric & underscore)
\W Matches any non-word character
\s Matches any whitespace character (spaces, tabs, line breaks)
\S Matches any non-whitespace character

quentifiers

Quantifier Description
* Matches 0 or more occurrences of the preceding element
+ Matches 1 or more occurrences of the preceding element
? Matches 0 or 1 occurrence of the preceding element
{n} Matches exactly n occurrences of the preceding element
{n,} Matches n or more occurrences of the preceding element
{n,m} Matches between n and m occurrences of the preceding element

position

Position Description
^ Matches the beginning of a string
$ Matches the end of a string
\b Matches a word boundary
\B Matches a non-word boundary

character class

Character Description
[abc] Matches a single character that is either a, b, or c
[^abc] Matches a single character that is not a, b, or c
[a-z] Matches a single character from a to z
[A-Z] Matches a single character from A to Z
[0-9] Matches a single digit from 0 to 9
[a-zA-Z] Matches a single character from a to z or from A to Z
[a-z0-9] Matches a single character from a to z or from 0 to 9
( com | dev | org) Matches any of the strings com, dev, or org

regexp type

regexp description
/pattern/ Matches the pattern
/pattern/i Matches the pattern case-insensitively
/pattern/g Matches the pattern globally
/pattern/m Matches the pattern over multiple lines
/pattern/s Matches the pattern over multiple lines
/pattern/y Matches the pattern in the sticky mode
/pattern/u Matches the pattern with unicode support

location object

Keyword Example Description
href location.href Returns the entire URL of the current page.
protocol location.protocol Returns the protocol of the current URL (e.g., http:, https:).
host location.host Returns the hostname and port number of the current URL.
hostname location.hostname Returns the hostname of the current URL.
port location.port Returns the port number of the current URL.
pathname location.pathname Returns the path of the current URL.
search location.search Returns the query string of the current URL.
hash location.hash Returns the anchor part of the current URL.
assign() location.assign('http://example.com') Loads a new document.
reload() location.reload() Reloads the current document.
replace() location.replace('http://example.com') Replaces the current document with a new one.

The getElementsByClassName() Method

//get all elements with a class containing "username" and "current", thought it
//doesn't matter if one is declared before the other

var allCurrentUsernames = document.getElementByClassName("username current");


//get all elements with a class of "selected" that exit in myDiv's subtree

var selected = document.getElementById("myDiv").getElementByClassName("selected");

The classList Property

  • add(value) - Adds the given string value to the list.If the value
    already exits, it will not be added.
  • contains(value) - indicates if the given value exits in the classList
    (true if so;false if not).
  • remove(value) - Removes the given string value from the list.
  • toggle(value) - If the value exits, remove it. If it does not exit, add it.

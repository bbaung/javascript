manipulating tables

Method Description
caption Pointer to the <caption> element (if it exists).
tBodies An HTMLCollection of <tbody> element
tFoot Pointer to the <tfoot> element (if it exists).
tHead Pointer to the <thead> element (if it exists).
rows An HTMLCollection of all rows in the table.
createTHead() Creates an empty <thead> element, places it into the table, and returns a reference
createTBody() Creates a <tbody> element, places it into the table, and returns a reference
createTFoot() Creates a <tfoot> element, places it into the table, and returns a reference
createCaption() Creates a <caption> element, places it into the table , and returns a reference
insertRow() Inserts a new row in the table at the specified index.
deleteRow() Deletes a row from the table at the specified index.
insertCell() Inserts a new cell in the row at the specified index.
deleteCell() Deletes a cell from the row at the specified index.

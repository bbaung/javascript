locating elements

Key Example Description
document.anchors <a name="anchorName"></a> Returns all anchor (a) elements in the document that have a name attribute.
document.applets <applet></applet> Returns all applet elements in the document. (Deprecated)
document.forms <form></form> Returns all form elements in the document.
document.images <img src="image.jpg" /> Returns all image (img) elements in the document.
document.links <a href="url"></a> Returns all link (a) elements in the document that have an href attribute.
